# FinalCleaner
FinalCleaner 的定位是轻巧美观，快捷方便并具有极客色彩的 Windows 垃圾清理软件。

## 开发技术

使用 C++ 开发可被 .NET Framework 托管的类库，使用 WinUI3 技术和 C# 语言开发 UI。

## 企划的程序模块

| 名称   | 描述                                                         |
| ------ | ------------------------------------------------------------ |
| Scout  | 卷管理库，能够向上提供卷描述，生成硬盘索引，卷事件处理器（例如设备插拔）等。 |
| WinUI3 | WinUI 界面，程序入口，同时也是各个类库的中控。               |
| 待定   | 开发中......                                                 |
